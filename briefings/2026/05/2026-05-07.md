  네이버는 수만 대 서버와 수백만 컨테이너에서 발생하는 막대한 메트릭 데이터를 실시간으로 안정적으로 처리하기 위해 오픈소스 시계열 DB인 VictoriaMetrics를 채택했다. 이 저장소는 12.5억 개 시계열과 555조 데이터포인트를 다루며, 수평 확장성과 무중단 장비전환 전략 등이 핵심이다.

  관련 링크: [네이버 D2: NAVER SECURITY SEMINAR](https://d2.naver.com/news/6236639)

- **스마트스토어센터 Oracle에서 MySQL로의 무중단 전환기**  
  스마트스토어 플랫폼 회원 파트가 Oracle DBMS에서 비용과 성능 문제로 인해 오픈소스 MySQL로 서비스 중단 없이 마이그레이션에 성공했다. 10년 이상 운영된 레거시를 대상으로 한 무중단 전환 전략과 기술적 해결 방안이 상세히 공유됐다.  
  관련 링크: [네이버 D2: 스마트스토어 MySQL 전환기](https://d2.naver.com/helloworld/6512234)

- **FE News 26년 4월: JavaScript Temporal API 표준 채택 임박**  
  JavaScript의 기존 Date 객체를 대체할 Temporal API가 9년간의 표준화 과정을 거쳐 TC39 Stage 4에 도달해 ES2026에 포함될 예정이다. 시간대, 불변값 처리 등 많은 개선점이 포함된 최신 API 소식.  
  관련 링크: [네이버 D2: FE News 26년 4월](https://d2.naver.com/news/6428522)
